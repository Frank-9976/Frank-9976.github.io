okay but what is this concept

Let $A$ be an abelian group. Let $\varphi \colon A^\omega \rightarrow A$ be a linear map from the direct product of countably many copies of $A$ to $A$ itself. We can regard $\varphi$ as an operation of countable arity. For the sake of having names, say that $(A, \varphi)$ is an "$\omega$-module". This is a dumb name, but it will have to do. Again, recall that this blog post is untitled.

okay, what now?

Given an $\omega$-module $(A, \varphi)$, say that $\varphi$ is "commutative" if the following identity is satisfied:

$\varphi(\varphi(x_{00}, x_{01}, x_{02}, \ldots), \varphi(x_{10}, x_{11}, x_{12}, \ldots), \varphi(x_{20}, x_{21}, x_{22}, \ldots), \ldots)$ = $\varphi(\varphi(x_{00}, x_{10}, x_{20}, \ldots), \varphi(x_{01}, x_{11}, x_{21}, \ldots), \varphi(x_{02}, x_{12}, x_{22}, \ldots), \ldots)$

That is, this identity holds for all families $(x_{ij})$ of elements of $A$ indexed by $\omega \times \omega$. Again, for the sake of having names, say that $(A, \varphi)$ with commutative $\varphi$ is a "commutative $\omega$-module".

do you actually have a motivation for this concept

Okay so, if we replace $\omega$ with some finite set $n = \{0, 1, 2, \ldots n-1\}$, then a commutative $n$-module is precisely a $\mathbb{Z}[f_0, f_1, \ldots, f_{n-1}]$-module. In fact, we can reduct (i.e. this destroys a lot of information) a commutative $\omega$-module to a $\mathbb{Z}[f_0, f_1, \ldots]$-module by defining

$f_0 x := \varphi(x, 0, 0, 0, \ldots),$
$f_1 x := \varphi(0, x, 0, 0, \ldots),$
$f_2 x := \varphi(0, 0, x, 0, \ldots),$

so on and so forth. The commutativity condition also guarantees similar things to the commutativity of a base ring. For instance, define an $\omega$-module homomorphism $\ell \colon (A, \varphi_A) \rightarrow (B, \varphi_B)$ as a linear map satisfying

$\ell(\varphi_A(x_0, x_1, \ldots)) = \varphi_B(\ell(x_0), \ell(x_1), \ldots)$

for all families $(x_i)$ of elements of $A$ indexed by $\omega$. Then the set of $\omega$-module homomorphisms from $A$ to $B$ should itself have an $\omega$-module structure. I'm not sure if this is significant at all, as I have not actually utilized this property while thinking about commutative $\omega$-modules. But, it's certainly where my motivation came from.

okay, what now? (again)

The kicker is that I've actually proven an (in my opinion, beautiful) theorem that confines how wild a commutative $\omega$-module can be. To state this theorem, I need yet another untitled concept, but this one is related to extension by $0$. In particular, given a subset $C \subseteq \omega$, we can view $A^C \subseteq A^\omega$ via "extension by $0$". In other words, we can fill in blank entries with $0$'s. Thus, we can restrict $\varphi$ to $A^C$. In fact, we did this earlier to produce the reduct, by letting $C$ be a singleton. Again, again, for the sake of having names, notate this restriction as $\varphi_C$. Then, I claim, for any commuative $\varphi$, we have

$$\bigcap_{C \text{ cofinite}} \operatorname{im} \varphi_C \neq A.$$
In other words, $\varphi$ must be "eventually non-surjective"!

whoa, that's so cool! tell me how you proved it, oh wise one

Firstly, thanks for the flattery. Secondly, we may prove the contrapositive, i.e., there must exist a witness $x = (x_{ij})$ to $\varphi$ failing to be commutative. We can construct it "one row at a time": To start, let $y = (y_i)$ be a family such that $\varphi(y) \neq 0$. Then, let $(x_{0j})$ be a family such that $x_{00} = 0$ and $\varphi(x_0) = y_0$. Then, let $(x_{1j})$ be a family such that $x_{10} = -x_{01}$, $x_{11} = 0$, and $\varphi(x_1) = y_1$. Continue on in this way, each time using that $\varphi_C$ is surjective for cofinite $C$. Eventually, we construct $x$ such that $\varphi(\varphi(x)) \neq 0$, but $(x_{ij}) + (x_{ji}) = 0$, (i.e. $x + x^T = 0$), so if $\varphi$ were commutative, $x$ would be in its kernel! (Alternatively, we could have constructed $x$ by alternating adding rows and columns, but abusing linearity like this simplifies notation a little.)

okay, what's the catch?

Oh yeah, er... this inequality is very sharp. The dumbest examples of commutative $\omega$-modules are constructed as follows: let $(A, \varphi)$ be any $\omega$-module, i.e. it can be non-commutative. Let

$$0 \rightarrow A \rightarrow B \rightarrow A \rightarrow 0$$
be a short exact sequence of abelian groups. We have the composition

$$\varphi' \colon B^\omega \rightarrow A^\omega \rightarrow A \rightarrow B$$
which has the same image as $\varphi$; we also have $\varphi'(\varphi'(x)) = 0$ for all $x$, so $\varphi'$ is commutative. (Note that this construction can never produce an example where $A = B \neq 0$.)

both dumb and non-dumb concrete examples

  • Let $A = \mathbb{Z}/p$ for some prime $p > 0$. $\omega$-module structures on $A$ in general can be very wild because of field theory nonsense. However, because commutative $\omega$-modules must be eventually non-surjective, we have that $\varphi$ must actually have finite support! Interestingly, one can show these are precisely the "constructable maps" in the sense that you don't need the axiom of choice to prove they exist. Alternatively, they're precisely continuous maps, putting the product topology on $A^\omega$.
  • However, the fun does not last. If we let $A = \mathbb{Z}/4$, using the "dumb" construction earlier, we can get very non-continuous but commutative $\omega$-modules on $A$. This also works on $(\mathbb{Z}/2)^2$, etc. I wish I knew how to fix this issue, but I don't have any natural ideas yet.
  • Interestingly, any $\omega$-module structure on $\mathbb{Z}$ is continuous. This is group theory folklore, if you're curious about the details, search up the "Baer-Specker group".
  • Here's possibly the most interesting example: suppose that

    $$\bigcap_{C \text{ cofinite}} \operatorname{im} \varphi_C = 0$$
    so that we have

    $$A \cong \projlim_{C \text{ cofinite}} \operatorname{coker} \varphi_C.$$
    Then we can actually describe such $\omega$-module structures via the universal property of projective limits. In particular, $\varphi$ composed with a projection to any particular $\operatorname{coker} \varphi_C$ must be finitely supported. However, as we look at weaker and weaker projections, the support can grow unbounded. Thus, we get interesting examples, such as $\varphi$ being a "weighted sum" of, say, $p$-adic integers.
  • Of course, who doesn't want to look at the free object of a category? Because our operation here has countable arity, our syntax trees can actually have depth indexed by $\omega_1$, the first uncountable ordinal. That being said, although I haven't checked the fine details, I'm pretty sure one can still abstractly define the free $\omega$-module in terms of operations in the signature of abelian groups with $\varphi$ added. We can also freely mod out by the commutativity condition to get the free commutative one. I believe the free (non-commutative) $\omega$-module on one generator can be viewed as a subring of

    $$\projlim \mathbb{Z}[f_0] \leftarrow \mathbb{Z}[f_0, f_1] \leftarrow \ldots$$
    with each projection sending $f_n$ to $0$; we have $\varphi(x) = f_0 x_0 + f_1 x_1 + \ldots$, and we take the smallest subring closed under $\varphi$. However, I haven't checked all the details.
